## The NŪR Protocol

**NŪR** (Nexus of Universal Resonance) is the foundational protocol for Star Dust frequency analysis.

### Mathematical Foundation

```
Resonance Score (RS) = ((TA + CF + SS) / 3) / RF × 10

Where:
  TA = Truth Alignment     (0.0 - 1.0)
  CF = Cultural Fit        (0.0 - 1.0)
  SS = Strategic Synergy   (0.0 - 1.0)
  RF = Resistance Factor   (0.0 - 1.0, lower is better)
```

**Special Cases:**
- If RF = 0: RS = 10.0 (perfect resonance, zero resistance)
- If RF = 1.0: RS approaches 0 (maximum resistance blocks resonance)

#### 3. Strategic Synergy (SS)
**Purpose**: Assesses mutual strategic value and growth potential

**Evaluation Criteria:**

- **Technology Synergy**: Complementary tech capabilities
- **Brand Alignment**: Compatible brand positioning and values
- **Resource Complementarity**: Assets each party brings
- **Growth Potential**: Scalability and expansion opportunities

#### 4. Resistance Factor (RF)
**Purpose**: Identifies barriers, friction points, and implementation challenges

**Evaluation Criteria:**
- **Competitive Conflicts**: Existing partnerships with competitors
- **Legal/Regulatory**: Compliance challenges, IP concerns
- **Technical Barriers**: Integration complexity, compatibility issues
- **Organizational Alignment**: Internal stakeholder buy-in
- **Resource Constraints**: Budget, time, personnel limitations

**Calculation (Lower is Better):**
```python
def calculate_resistance_factor(opportunity):
    """
    Calculate resistance factor (barriers to partnership)
    Lower scores indicate fewer barriers
    """
    conflicts = assess_competitive_conflicts(opportunity)
    legal = assess_legal_regulatory_issues(opportunity)
    technical = assess_technical_barriers(opportunity)
    organizational = assess_organizational_challenges(opportunity)
    resources = assess_resource_constraints(opportunity)
    
    return (conflicts + legal + technical + organizational + resources) / 5.0

# Result: 0.25 (relatively low resistance)
```

## Implementation

### Python Integration

See `resonance_calculator.py` for the reference implementation:

```python
from dataclasses import dataclass

@dataclass
class ResonanceScore:
    """NŪR-based resonance scoring"""
    truth_alignment: float
    cultural_fit: float
    strategic_synergy: float
    resistance_factor: float
    
    @property
    def overall_score(self) -> float:
        """Calculate overall resonance score"""
        if self.resistance_factor == 0:
            return 10.0
        return ((self.truth_alignment + self.cultural_fit + 
                 self.strategic_synergy) / 3.0) / self.resistance_factor * 10
    
    @property
    def recommendation(self) -> str:
        """Get partnership recommendation"""
        score = self.overall_score
        if score >= 9.0:
            return "PERFECT_RESONANCE"
        elif score >= 8.0:
            return "HIGH_PRIORITY"
        elif score >= 6.0:
            return "GOOD_FIT"
        elif score >= 4.0:
            return "MODERATE_POTENTIAL"
        elif score >= 2.0:
            return "LOW_RESONANCE"
        else:
            return "MISALIGNED"

# Usage
score = ResonanceScore(
    truth_alignment=0.85,
    cultural_fit=0.90,
    strategic_synergy=0.88,
    resistance_factor=0.25
)

print(f"Overall Score: {score.overall_score:.2f}/10")
print(f"Recommendation: {score.recommendation}")
```
