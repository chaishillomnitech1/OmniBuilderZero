## Quick Start: Agentic Patterns

```python
# Initialize the adaptive learning environment
from agentic_patterns.learning import AdaptiveLearningEnvironment
from agentic_patterns.core import LearningStyle

# Create environment and start session
env = AdaptiveLearningEnvironment()
session = env.start_session(
    learner_id="student_001",
    name="Alex",
    age=8,
    interests=["space", "dinosaurs"],
    learning_style=LearningStyle.VISUAL
)

# Learn something new
response = env.learn("student_001", "addition")
print(response.content)

# Get personalized recommendations
recommendations = env.get_recommendations("student_001", "math")

# End session with summary
summary = env.end_session("student_001")
```
