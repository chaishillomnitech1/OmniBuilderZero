## 🚀 Quick Start

### Prerequisites
- Node.js v16+ (v18+ recommended)
- npm v8+
- Git

### Setup (3 minutes)
```bash
# Clone the repository
git clone https://github.com/chaishillomnitech1/OmniBuilderZero.git
cd OmniBuilderZero

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your settings (private keys, RPC URLs, API keys)

# Compile contracts
npm run compile

# Run tests
npm test
```

### Deploy to Testnet
```bash
# Deploy to Sepolia
npm run deploy:sepolia

# Deploy to Scroll Sepolia
npm run deploy:scroll-sepolia
```
